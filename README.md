# Expense Tracker with Budget Alerts

## Description
This project is a real-time expense tracker that allows users to add expenses and analyze spending patterns. It includes budget alerts and data visualization.

## Features
- Add daily expenses
- Monthly and daily budget alerts
- Category-wise spending analysis
- Data stored permanently using CSV files
- Graphical visualization

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Insights
- Helps track spending patterns
- Identifies high-expense categories
- Alerts users when budget is exceeded
